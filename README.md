# Pirate Radio Issue Tracker

Welcome to the official issue tracker for the **Pirate Radio** projects.

**Please Note:** This repository is used *exclusively* for tracking bugs, managing feature requests, and collecting feedback. 

---

## How to submit an Issue

All project-related tasks, bugs, and discussions happen in the "Issues" tab which can be found here: [Pirate Radio Issues](./issues)

Before opening a new issue, please **search the existing issues** to see if your problem or suggestion has already been reported to reduce duplicates.

---

## How to File a Good Report

When you submit a new issue, please provide as much detail as possible to help us resolve it quickly.

* **For Bug Reports:**
    * A clear and descriptive title.
    * Steps to reproduce the bug.
    * What you expected to happen.
    * What *actually* happened (include screenshots or error messages if possible).
    * Your device, operating system, and application version.

* **For Feature Requests:**
    * A clear description of the new feature.
    * An explanation of *why* this feature would be valuable (e.g., the problem it solves).
    * (Optional) Any mockups or examples of how it might work.
